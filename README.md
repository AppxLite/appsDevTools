# appsDevTools
appsDevTools for Wordpress plugin.

基于 wordpress 定制小程序和公众号的开发插件，主要实现小程序api所需的功能和公众号自动回复功能实现，让wordpress网站+公众号+小程序实现数据互通。

使用方法：下载插件》安装插件》开启设置
